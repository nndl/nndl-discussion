# 蒲公英书系列讨论区

[蒲公英书系列](https://github.com/nndl)（[邱锡鹏](https://xpqiu.github.io/) 的神经网络与深度学习系列教材）的社区讨论入口。

请到 [**Discussions**](https://github.com/nndl/nndl-discussion/discussions) 发帖，话题包括：

- **Announcements**：新版发布、勘误汇总、活动通知
- **Q&A**：章节内容、概念理解、习题与思考题的提问与回答
- **General**：读书心得、学习路径、教学经验交流
- **Ideas**：对教材内容、配套资源、可视化工具的建议

## 不在这里处理的事

- **某本书的勘误**：到对应书仓库提 Issue
  - 第 2 版 / 通识版：[`nndl/nndl`](https://github.com/nndl/nndl)
  - 案例与实践：[`nndl/nndl-practice`](https://github.com/nndl/nndl-practice)
  - 大模型与智能体：[`nndl/llm-agent`](https://github.com/nndl/llm-agent)
- **可视化资源**：https://nndl.github.io/viz/
- **系列主站**：https://nndl.github.io

## 第 1 版习题答案

第 1 版（2020）的课后习题讨论保留在本仓库的 [Issues](https://github.com/nndl/nndl-discussion/issues) 中（每题一个 issue），统一打上 `第一版` 标签。新版相关的提问、讨论请改到 Discussions。
